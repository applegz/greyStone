# Loan Amortization app

This app allows clients to

- Create a user
- Create loan
- Fetch loan schedule
- Fetch loan summary for a specific month
- Fetch all loans for a user
- Share loan with another user

## To Run

`uvicorn main:app --reload`

## To Test

`pytest`

`http://127.0.0.1:8000/docs#/`
